Выполнить задания из консоли и текстового файла информации, такой как текст, числа и контролировать корректность
вводимых данных, а также  контролируя состояние потоков ввода/вывода. При возникновении ошибок, связанных с корректностью
выполнения математических операций, генерировать и обрабатывать исключительные ситуации. Предусмотреть обработку исключений,
возникающих при нехватке памяти, отсутствии требуемой записи (объекта) в файле, недопустимом значении поля и т. д.
